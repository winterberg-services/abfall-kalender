Kalender auf Android abonnieren

Android unterstützt leider keine direkten Kalender-Abos. So geht's trotzdem:

  1. 1. URL kopieren (Button unten)
  2. 2. Google Kalender öffnen (Web oder App)
  3. 3. Einstellungen → Kalender hinzufügen → Per URL
  4. 4. URL einfügen und abonnieren

Tipp: In Google Kalender den Link mit https:// statt webcal:// einfügen.
